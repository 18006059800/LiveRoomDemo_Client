# LiveRoomDemo(移动端)
> 这是一个用Vue.js写的一个直播间Demo,主要实现了以下功能

* 拉取服务器上的直播流(移动端拉取hls流、电脑端拉取rtmp流)
* 基于websocket的直播聊天室
* 炫丽的直播间弹幕
* 直播间的实时数据统计



# How to Use

- quick start

``` bash
git clone https://github.com/wangdahoo/vonic-webpack-starter.git
cd vonic-webpack-starter
npm install -g yarn
yarn
```

- development

```bash
npm run dev
```

- build

```bash
npm run build
```
